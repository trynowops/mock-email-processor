## Installing and Running

This is a Spring Boot app. Setup is identical to [this guide](https://spring.io/guides/gs/spring-boot/):

1. Clone the repo.
2. Ensure you have Java 17 available.

```shell
java -version
```

3. Run `gradlew` to start.

```shell
./gradlew bootRun
```
4. SIGINT (`ctrl+c`) to stop.
